Proyectos de Electrónica

Módulo de Relé de 5V con Optoacoplador

El objetivo principal de este proyecto es demostrar cómo controlar de forma segura cargas de alto voltaje o potencia utilizando un microcontrolador o cualquier otra señal de bajo voltaje, sin el riesgo de dañar el circuito de control.

Tecnologías: Proteus
Módulo de Relé de 5V SMD

Diseño completo de un módulo de relé de 5V con componentes de montaje superficial (SMD). El proyecto incluye los archivos de simulación, el esquema eléctrico y el diseño de la placa de circuito impreso (PCB). Es ideal para quienes buscan diseñar módulos compactos y eficientes para proyectos de electrónica.

Tecnologías: Proteus
Regulador de Voltaje Ajustable con LM317

El circuito se basa en el LM317, un regulador de voltaje lineal ampliamente conocido que permite ajustar la tensión de salida. Este diseño es muy útil para proyectos que requieren diferentes niveles de voltaje, como alimentar microcontroladores, sensores o pequeños motores.

Tecnologías: KiCad
Monitor Biométrico Portátil

Dispositivo electrónico que integra un sensor biomédico MAX30102, una pantalla OLED SSD1306 y un Arduino Nano. El objetivo principal del proyecto es ofrecer una plataforma de hardware de código abierto para el monitoreo de la frecuencia cardíaca y el nivel de oxígeno en sangre (SpO₂).

Tecnologías: KiCad
Módulo de LEDs SMD

Módulo electrónico compacto de iluminación basado en LEDs de montaje superficial (SMD). El proyecto fue desarrollado en Altium Designer para aplicaciones de señalización, indicadores o iluminación de bajo consumo.

Tecnologías: Altium
Fuente de Alimentación Regulada de 5V

Diseño para una fuente de alimentación regulada de 5V basada en el circuito integrado LM7805 y es una solución robusta y confiable para alimentar prototipos electrónicos, microcontroladores y otros dispositivos digitales.

Tecnologías: Altium
Control de Velocidad con LM555

Ddiseño para un control de velocidad basado en el circuito integrado LM555 y es una solución robusta y versátil para controlar la velocidad de un motor de corriente continua (DC), actuadores o la intensidad de una carga a través de la modulación por ancho de pulso (PWM).

Tecnologías: Altium
